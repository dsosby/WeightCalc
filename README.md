# weightcalc

A simple weight calculator written in Clojure as a learning project.

## Usage

```
lein run
```

Accepts a workout in key/value pairs, e.g.:

```
lein run squat 210 bench 155 deadlift 305
```

Currently supports the following exercises:
* squat
* bench
* deadlift
* press
* row
